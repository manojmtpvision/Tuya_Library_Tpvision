## 功能概述

涂鸦智能摄像机面板SDK（简称：TuyaCameraPanelSDK）为方便用户，减少摄像头面板开发成本，提供了涂鸦摄像头相关的面板，主要包括：

- Rn面板(详见 [TuyaPanelSDK文档](https://tuyainc.github.io/tuyasmart_panel_android_sdk_doc/))
- 原生native面板
- 普通门铃接听面板  （界面显示：实时图片+接听挂断）
- 视频流门铃接听面板（界面显示：实时视频流+接听挂断）
- 云存储面板
- 回放面板
- 设置面板
- 消息中心面板
- 本地相册面板
- 实现自定义面板
    - 自定义回放面板
    - 自定义云存储面板
    - 自定义本地相册面板
    - 自定义消息中心面板
    - 自定义设置面板
        - 自定义修改设备名称/头像
        - 自定义意见反馈
